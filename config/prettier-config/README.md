## ‚ú® Getting started

To install the `@draconic/prettier-config` constant in your project, use the package manager of your choice:

Using [npm](https://www.npmjs.com/):

```shell
npm i -D @draconic/prettier-config
```

Using [yarn](https://yarnpkg.com/en/):

```shell
yarn add -D @draconic/prettier-config
```

Using [pnpm](https://pnpm.io/):

```shell
pnpm add -D @draconic/prettier-config
```
